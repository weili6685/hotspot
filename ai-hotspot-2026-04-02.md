## 科技公司最新动态

### Anthropic
- **Claude Code 源码泄露事件（2026-03-31）**：npm 发布时意外包含 .map 源码文件，泄露完整内部实现及未发布功能（KAIROS 自主代理模式）。Anthropic 已确认并修复。
- Claude Code v1.0.0 已正式 GA，生态已超 600 个社区工具，当前 82k+ GitHub stars。
- Claude Opus 4.6 在 Terminal-Bench 2.0 得分 58.0%，仍是 Cursor Composer 2 的主要对标。

### OpenAI
- **收购 Windsurf（~$3B）**：为应对 Anthropic 和 Google 在 AI 编程工具领域的竞争压力，OpenAI 收购 AI 编程工具 Windsurf。
- 收购动机：巩固企业级 AI 代理编程市场地位，应对 Cursor、Claude Code 的快速崛起。

### Cursor / Anysphere
- **Composer 2 发布（2026-03-19）**：自研代码专用模型，Terminal-Bench 2.0 得分 61.7%（超越 Claude Opus 4.6 的 58.0%），价格仅 $0.50/M tokens（Opus 的 1/10）。
- Anysphere 年化收入已达 **$20 亿**，估值谈判据报达 **$500 亿**。

### Google
- Anthropic 和 Google 的模型在编程基准测试中持续领先 OpenAI，成为开发者首选。
- Google 持续投资 Anthropic，Claude 是其 AI 合作战略核心。

### Windsurf（已被 OpenAI 收购）
- 2025 年曾被 Anthropic 限制直接访问 Claude 模型，是促成 OpenAI 收购谈判的背景之一。
