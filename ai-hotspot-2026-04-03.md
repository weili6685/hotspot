## 科技公司最新动态

### Anthropic
- **Claude Code 源码泄露事件**：npm 打包错误导致源码 map 文件公开，Anthropic 官方确认。泄露揭示多项未发布功能：
  - **KAIROS**：持续后台 Agent，可自主周期性修复错误、执行任务、发送推送通知
  - **Dream Mode**：Claude 后台持续思考、迭代想法
  - **Undercover Mode**：向开源仓库进行"隐身"贡献的模式
  - **frustration regex**：检测用户挫败情绪的正则表达式机制
- Claude Code 累计 78,400+ GitHub stars（Trendshift 显示 4 月增量 13,429）

### Cursor (Anysphere)
- **2026-03-19 发布 Composer 2**：首个编程工具公司自研前沿级 AI 模型
  - Terminal-Bench 2.0 得分 61.7%，超越 Claude Opus 4.6（58.0%）
  - 定价 $0.50/M tokens，仅为 Anthropic 旗舰模型的 1/10
  - 2026 年初 ARR 突破 $20 亿
- 来源：[Bloomberg](https://www.bloomberg.com/news/articles/2026-03-19/ai-coding-startup-cursor-plans-new-model-to-rival-anthropic-openai)

### OpenAI
- **$30 亿收购 Windsurf**：OpenAI 以 30 亿美元收购 AI 原生 IDE Windsurf（年收入仅约 4000 万），反映对抗 Google 和 Anthropic 的紧迫性
- 来源：[VentureBeat](https://venturebeat.com/ai/openais-3b-windsurf-move-the-real-reason-behind-its-enterprise-ai-agent-code-push)

### AI 编程工具竞争格局
- Cursor、Windsurf、GitHub Copilot、Claude Code、Aider 从"谁用的模型更好"转向"谁的上下文更大、执行循环更强"
- Vibe Coding 已成主流范式，但安全漏洞问题引发业界广泛讨论
