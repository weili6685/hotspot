## 科技公司最新动态

### Anthropic
- **Claude Code 源码泄漏确认**（2026-04 初）：因 npm 打包失误导致 source map 文件公开，Anthropic 官方已确认事件。泄漏揭露了 KAIROS 后台 Agent 功能（无人工干预下自动修复错误/执行任务）、Undercover Mode（向开源项目隐身贡献）等未公开特性。
- **Claude + Codex 进入 GitHub Copilot**（2026-02-04）：Claude 和 Codex 现已作为 Copilot Pro+ 和 Copilot Enterprise 的 Coding Agent 可用。
- **OAuth 保护加固**（2026-01-09）：部署服务端保护阻止未授权 OAuth token 访问，封堵了 OpenCode（原 OpenClaw）通过伪造 HTTP headers 使用 Claude 模型的漏洞。

### OpenAI
- **Windsurf 收购落空**：OpenAI 原计划以 ~30 亿美元收购 Windsurf，但与微软在 IP 权利上产生冲突，最终未能成交。Windsurf 被三拆：Google DeepMind 签署 24 亿美元许可协议并招募 CEO/联创；Cognition（Devin 母公司）收购 IP、产品、品牌及约 210 名员工；OpenAI 最终空手而归。
- **Codex Agent**：现已整合进 GitHub Copilot Pro+ 生态。

### Google / DeepMind
- **Windsurf 人才与技术收割**：以 24 亿美元许可协议拿下 Windsurf 技术，并招募 CEO Varun Mohan 和联创 Douglas Chen 加入 DeepMind。

### Cursor
- **Cursor 3 发布**（2026-04 初）：代号 "Glass"，全新 Agent-First 编码界面，以自然语言描述任务交由 AI Agent 自主完成。估值达 293 亿美元。
- **竞争压力加剧**：Anthropic 和 OpenAI 提供 200 美元/月订阅（含超过 1000 美元用量价值），部分开发者从 Cursor 转向 Claude Code 和 Codex。
- **自研模型计划**（2026-03-19，Bloomberg）：Cursor 计划自研 AI 模型以对抗 Anthropic 和 OpenAI。

### Windsurf（被分拆）
- 原公司已不复存在，品牌和产品被 Cognition 接管，核心技术和团队被 Google DeepMind 吸纳。
