## 科技公司最新动态

### Anthropic
- **Claude Code 源码泄露（2026-03-31）**：因 npm 打包错误，完整源码（含注释和架构细节）随 `.map` 文件一同泄露，Anthropic 官方确认。泄露代码中发现神秘功能开关 **KAIROS**，疑似未发布的自主 Agent 模式。
- **Claude Code 定价策略**：$200/月套餐预估消耗 $5,000/年算力资源，相当于 25:1 的补贴比例，Anthropic 实际每位用户补贴约 $24,800。
- **生态扩张**：Claude Code 全源码于 2026-03-31 公开，GitHub stars 突破 82K，社区生态爆发式增长。

### Cursor (Anysphere)
- **Composer 2 发布（2026-03-19）**：推出完全自研的代码专用模型，定价 $0.50/M tokens（约为 Anthropic 旗舰模型的 1/10）。正在以约 **$500 亿估值**筹备新一轮融资。

- 收购 Windsurf 的谈判最终落空，Windsurf 于 2025 年 7 月三方拆解：Google DeepMind 签署 $24 亿授权协议并挖走 CEO；Cognition（Devin 母公司）收购 IP、产品及 210 名员工。
