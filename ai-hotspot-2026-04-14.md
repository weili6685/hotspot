## 科技公司最新动态

### Anthropic
- **Claude Code 源码泄露事件**：因 npm 包含 source map 文件导致源码泄露，已确认并修复。泄露代码中发现 KAIROS（后台自主 Agent）和 Dream Mode（后台思考）等未发布功能
- **Claude Code GitHub 突破 82K+ star**，社区生态持续爆发
- Claude Opus 4.6 在 Terminal-Bench 2.0 得分 58.0%

### Cursor (Anysphere)
- **Composer 2 发布**（2026-03-19）：自研 code-only 模型，Terminal-Bench 2.0 得分 61.7% 超过 Claude Opus 4.6
- 定价 $0.50/M token，仅为 Anthropic 旗舰的 1/10
- DAU 超 100 万，服务 5 万家企业（含 Stripe、Figma）
- 正在进行新一轮融资，估值约 $500 亿

### OpenAI / Windsurf
- **Windsurf 三方拆分**：Google DeepMind 签下 $24 亿授权协议并挖走 CEO Varun Mohan；Cognition（Devin）收购 Windsurf IP/产品/品牌及约 210 名员工；OpenAI 空手而归
- OpenAI 原计划 $30 亿收购 Windsurf 因微软 IP 权利争议告吹

### Google
- Google DeepMind 通过 Windsurf 交易获得 AI 编码领域关键人才和技术授权
