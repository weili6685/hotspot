## 科技公司最新动态

### Anthropic
- **Claude Mythos** 新模型发现数千个跨主流系统的零日漏洞，安全能力大幅跃升
- **Claude Code 源码泄露事件**（3/31）：npm 打包错误暴露完整源码，泄露中发现未发布的 **KAIROS** 自主 Agent 模式和下一代模型内部代号 **Capybara**
- Claude Code GitHub star 突破 **82K**，持续领跑开发者工具生态

### Cursor
- 正在开发 **Composer 2**——可执行长时间编码任务的 AI Agent
- 计划训练自有模型对标 Anthropic / OpenAI（[Bloomberg 报道](https://www.bloomberg.com/news/articles/2026-03-19/ai-coding-startup-cursor-plans-new-model-to-rival-anthropic-openai)）
- 融资估值约 **$50B**

### Windsurf (原 Codeium)
- 2025年7月被三方瓜分：Google DeepMind $2.4B 签约 + 聘用 CEO；Cognition 收购 IP/产品/品牌及约210名员工；OpenAI 空手而归

### OpenAI
- $3B 竞购 Windsurf 失败后转向与 **OpenCode** 合作集成 Codex
- OpenCode 移除 Claude OAuth 支持，切换至 OpenAI Codex / GitHub / GitLab

### Google
- DeepMind 通过 Windsurf 交易获得核心团队，加强 AI 编码工具布局
