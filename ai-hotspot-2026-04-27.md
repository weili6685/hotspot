### Anthropic
- **Claude Code 源码泄露**：通过 npm 打包失误意外暴露，安全研究员 Chaofan Shou 在 X 上曝光，累计 2880 万浏览
- 泄露内容揭示 **KAIROS** 功能：支持 Claude Code 作为持久后台 Agent，自主定期修复错误、运行任务、推送通知
- **dream 模式**：允许 Claude 在后台持续思考并迭代想法
- Claude Mythos 安全模型：已在主要系统中发现数千个零日漏洞
- Anthropic 在编码基准上超越 OpenAI，正成为开发者首选

### OpenAI
- **Codex 大升级**：新增桌面控制能力（对标 Claude Code），内置浏览器，让 Agent 可执行更复杂指令
- Windsurf 收购疑云：OpenAI 以 30 亿美元收购 Windsurf，Anthropic 随即限制其 Claude 模型访问权限
- Codex 市场认知：65.3% Reddit 用户倾向 Codex，但讨论热度远低于 Claude Code

### Google
- 最新 TPU 正式推出（2026-04-23），为 Anthropic、OpenAI、Meta 等提供算力支撑
- AI 模型在编码基准上表现超越 OpenAI，助力 Anthropic 等合作方
