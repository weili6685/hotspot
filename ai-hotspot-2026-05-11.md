## 科技公司最新动态

### Anthropic
- **Claude Code 持续领跑**：SWE-bench Verified 实测 80.8%，业内最高。
- **Claude + Cowork** 推出远程操控 Mac / 桌面能力，agent 进入"操作系统级"操作阶段。
- **Claude Mythos** 在 2026/4 被披露：发现多个主流系统的 zero-day（见 The Hacker News 报道）。
- 官方仓库 `anthropics/financial-services` 当日 +1449 star，登顶日榜。

### OpenAI
- **Codex 全面升级**：新增桌面控制权限与 in-app browser，正面对标 Claude Code（TechCrunch, 2026-04-16）。
- **Windsurf 收购落空**：OpenAI 原计划 30 亿美元收购 Windsurf，被 Microsoft 在 IP 上卡住后崩盘。

### Google
- **Stitch（AI 设计画布）**于 2026/3 全面重启：无限画布 + context-aware design agent + 即时原型。
- **24 亿美元 license-and-acquihire 拿下 Windsurf 核心团队**（CEO Varun Mohan、co-founder Douglas Chen 及约 40 名核心工程师）。

### Cursor (Anysphere)
- 正在以 **~500 亿美元估值融资 20 亿+ 美元**，a16z + Thrive 领投，Nvidia 战略跟投。
- ARR 飞涨：2025/01 1 亿 → 2025/06 5 亿 → 2025/11 10 亿 → 2026/02 20 亿美元。
- 自研模型计划已公开（Bloomberg 2026-03-19），准备对线 Anthropic / OpenAI。

### Windsurf / Cognition
- 残余 IP / 品牌 / 约 210 人被 **Cognition（Devin 母公司）** 以约 2.5 亿美元收购。
- **Windsurf 2.0**（Agent Command Center + Devin 集成）于 2026/4 发布。
